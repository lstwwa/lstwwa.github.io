卸载与安装系统应用

以下有两种常用的方式来进行对系统软件的卸载和停用

  • 停用软件

      adb shell pm disable-user +APPpackagename

  • 启用

      adb shell enable +APPpackagename

  • 卸载软件

      adb shell pm uninstall --user 0 +APPpackagename

  • 安装

      adb shell pm install-existing --user 0 +APPpackagename

他们有以下区别

  • 禁用的话接口会依然存在，但如果卸载的话就不会有接口

    例如华为的下拉搜索，使用禁用的话依然保留下拉动画

  • 某些情况下卸载反而会比停用更加安全

    例如小米禁用或者卸载应用商店后会卡米，但是禁用状态下无法进入ADB调试而卸载则可以进入ADB

注意

  • 1、我建议你先对重要数据进行备份后再开始你的精简操作，对手机的ADB调试在一些情况下拥有足够的危险性

  • 2、如果你对以下列表表示怀疑请自行利用软件进行核对并且不要使用本列表的任何数据，当然你可以使用Devcheck查看本机包名

  • 3、在一些系统上比如拥有隐私空间或者第二个用户，使用卸载的方式精简依赖比如谷歌服务，可能会导致一些很奇怪的问题，请自行测试

  • 4、我仅列出包名列表但使用本列表做的所有操作导致数据丢失本人不负任何责任

  • 5、所有的列表都会有一定的时限性，本列表总结于2022年1月3日
