自定义命令行

使用mac自带的Terminal时，需要首先在用户目录下创建三个无后缀文件：

  • 第一个： .zlogin
  • 第二个： .bashrc
  • 第三个： .bash_profile

.zlogin 内容

if [ -f ~/.bashrc ]; then
source ~/.bashrc
fi
if [ -f ~/.bash_profile ]; then
source ~/.bash_profile
fi

.bash_profile 内容

export CLICOLOR=1
export LSCOLORS=Exfxcxdxbxegedabagacad

# Tell grep to highlight matches
export GREP_OPTIONS='--color=auto'

.bashrc 内容

alias nano='nano -c'
alias c='clear'
alias python='python3'
alias l='ls -l'
alias la='ls -la'

安装brew

/bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)"

Mac软件已损坏

sudo xattr -r -d com.apple.quarantine +APPNAME

APPNAME请在访达中拖动

删除Microsoft AutoUpdate

目录如下，可以使用访达前往文件夹。

/Library/Application Support/Microsoft/

找到“MAU”或“MAU2.0”后将“Microsoft AutoUpdate.app”拖到废纸篓即可。
