    判断题一

  • 1.多级耦合的调谐放大器的通频带比组成它的单级单调谐放大器的通频带宽。 （×）

  • 2.鉴频灵敏度是指在调频信号的中心频率 附近单位频偏所产生输出电压的大小，越大越好。（√）

  • 3.LC并谐振电路谐振时，感抗与容抗都为零，所以回路呈纯电阻特性。( × )

  • 4．当 较大时，调频波被认为是恒定带宽的调制。 （ √ ）

  • 5.功率放大器是大信号放大器，要求在不失真的条件下能够得到足够大的输出功率。（√）

  • 6.采用倍频器，可以降低发射机主振器的频率，有利于稳频、提高发射机的工作稳定性和扩展发射机的波段。( √ )

  • 7.在非线性电路中线性时变元件是线性的，所以适合线性叠加原理。 （ × ）

  • 8.放大器必须同时满足相位平衡条件和振幅条件才能产生自激振荡。 （ √ ）

  • 9.传输线变压器与普通变压器传递能量的方式是不同的。( √ )

  • 10.VCO电路是锁相环路和AFC电路的必要组成部分。 （ ×）

  • 11.电感三点式振荡器的输出波形比电容三点式振荡器的输出波形好。 （×）

  • 12.锁相环路就是通过输出信号相位跟踪输入信号相位，最终使两者相位差锁定为零( × )

  • 13.普通调幅电路在时域上是实现调制信号 与载波信号 的相乘运算；在频域上是将 的频谱不失真地搬移到载频的两端。 （ √ ）

  • 14.在调谐放大器的LC回路两端并上一个电阻R可以加宽通频带。 （√）

  • 15.电容三点式振荡电路输出的谐波成分比电感三点式的大，因此波形较差。( × )

  • 16.并联型石英晶体振荡电路的振荡频率处于 和 之间，石英谐振器等效为电感（ √ ）

  • 17.双边带（DSB）信号的振幅与调制信号的规律成正比。 （√）

  • 18.集总参数指元件参量与流经它的电流或加于其上的电压的数值无关。( × )

  • 19.在甲类、乙类、丙类功率放大器中，丙类功率放大器的效率最高。 （√）

  • 20.调频有两种方法，分别称为直接调频和间接调频。 （√）

  • 21.调频波的调频指数与调制信号幅度成正比，与调制信号频率成反比。(√)

  • 22.频率合成技术可以在一定频率范围内提供一系列高准确度和高稳定度的频率源（ √ ）

  • 23.锁相环路与自动频率控制电路实现稳频功能时，锁相环路的性能优越。 （√）

  • 24.谐振放大器工作在过压状态，现要调整使之工作到临界状态，若其他参数不变，可以增加负载Re来实现。( × )

  • 25.调频、调相、鉴频、鉴相电路属于非线性频率变换电路。 （ √ ）

  • 26.LC回路的品质因数Q值愈小，其选频能力愈强。 （×）

  • 27.克拉泼振荡电路比考毕兹振荡电路的频率稳定度高，是因为克拉泼电路的振荡回路中接入一个电容C3，从而能减小晶体管输入、输出电容对振荡回路的影响。(√)

  • 28.检波效率 只对检波电路有关，而与输入信号大小无关。 （√）

  • 29.混频电路又称变频电路，在变频过程中改变的只是信号的载频。 （×）

  • 30.单边带通信之所以得到广泛的应用，是因为它与普通调幅波相比大大节省了能量，而且占据频带压缩了。( × )

17、在甲类、乙类、丙类功率放大器中，丙类功率放大器的效率最高。 （ √ ）

18、LC回路的品质因数Q值愈小，其选频能力愈强。 （ × ）
