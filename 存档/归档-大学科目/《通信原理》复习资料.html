名词解释

24.信息传输速率

信息传输速率：每秒传输的信息量，单位Bit/S。

25.包络检波

包络检波：提取接收信号的幅度值从而获得系带数字信号的方式。

26.信道

信道：是指以传输媒介（质）为基础的信号通道。

27.码间串扰

码间串扰：是指由于噪声和干扰的影响，使喜好相邻马原的脉冲（出现拖尾叠加，在接收端造成判决困难的现象。

28.信源编码

信源编码：是为了提高数字信号的有效性以及为了使模拟信号数字化而采取的编码。

26.基带信号：

是指没有经过调制（频率搬移）的原始信号。

27.信息传输速率：

是指单位时间内传输的信息量。单位为bit/s。

28.噪声：

是通信系统中对信号有影响的所有干扰的集合。

29非均匀量化：

是一种在整个动态范围内量化间隔不相等的量化。在小信号时，量化级间隔小，而大信号时，量化时间隔大，以提高小信号的信噪比。

30.误码率：

是指错误接收的码元数在传送总码元数中所占的比例。

六、（本题10分）采用13折线A律编码，设最小的量化间隔为1个量化单位△，已知抽样值为－963△。
（1）试求此时编码器输出8位码组，并计算编码后的量化误差；
（2）试写出对应于该7位码(不包括极性码)的均匀量化11位码。

解：

  • （1）设输出码组为C1C2C3C4C5C6C7C8

Is = -963 < 0 ,所以 C1=0,又因为512 < Is < 1024,所以C2C3C4 = 110

963 - 512 = 451, V7=32, 所以451/3 = 14.1，所以C5C6C7C8 = 1110

所以C1C2C3C4C5C6C7C8 = 01101110

编码后的输出电平为：
Ic+V7/2 = ( 512 + 14 * V7 ) + V7/2 = (512 + 14 * 32) + 32/2 = 976V

编码后的量化误差为：Ic - Is = 976 -963 = 13V
7位码的11位码为：11110000000

七、（本题10分）采用13折线A律编码，最小量化间隔为1个量化单位，已知抽样脉冲值为-95量化单位，试求此时编码器输出码组，并计算量化误差（段内码采用均匀量化）。

解:

  • (1)

已知抽样脉冲值Is = -95，设码组的8位码分别为C1C2C3C4C5C6C7C8.

因为Is<0，故C1=0

又因为Is>64，且Is<128，故位于第4段，段落码C2C3C4=011

第4段内的量化间隔为4，由Is=64+7×4+3知，Is位于第4段第7量化级，C5C6C7C8=0111

因此，输出码组为C1C2C3C4C5C6C7C8 = 00110111

译码输出-(64+7×4+2/ 2)=-94,

量化误差为: -95-(-94)=-1 (单位)
