镜像的下载与制作

下载选项

官方的下载地址

https://www.raspberrypi.org/software/operating-systems/
  • 其中有三个选择与其他系统，这里只介绍官方系统：

    1:Raspberry Pi OS with desktop and recommended software
    2:Raspberry Pi OS with desktop
    3:Raspberry Pi OS Lite

  • 不会选择？

    如果是小白第一次接触树莓派，请选择，1
    如果已经使用过并且对功能了解，请选择，2
    如果无需桌面（比如Zero）可以选择，3

写入SD卡

格式化SD卡

  • 这里建议优先对SD卡进行格式化避免麻烦

    sdcardformatter下载地址：
    https://www.sdcard.org/downloads/formatter/

主网站进不去？这里是第三方网站：
https://www.sdcardformatter.com/download/

写入SD卡

  • 为了避免麻烦，这里可以拔出所有USB设备以防错误操作导致数据丢失。

    选择固件——>选择设备——>刷写
    https://sourceforge.net/projects/win32diskimager/files/Archive/

  • 还没结束，这时你将得到一个打不开的盘以及一个boot

  • 这时windows会提醒你格式化SD卡，请取消该选项框

新建ssh文件

  • 进入boot盘

    这里你需要创建一个
    ssh

  • 快速的创建方法是新建一个.txt文件，命名为ssh.txt后删除.txt
  • 如果你发现你的windows创建txt后后缀消失了，请点击文件夹选项打开显示后缀

如果没有屏幕，键盘，鼠标应该怎么办？

连接wifi或者连接网线，在网络连接中只有通过查看路由器或者通过nmap等工具的扫描可以得出ip，但是在树莓派没有网线接口或者需要wifi时，可以通过下列方法来连接网络。

  • 在boot分区中创建文件名字为
    wpa_supplicant.conf
  • 随后在该文件中写入以下内容
    country=CN
    ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
    update_config=1

    network={
    ssid=”WiFi-A” #你的Wi-Fi名字
    psk=”12345678” #你的Wi-Fi密码
    key_mgmt=WPA-PSK #Wi-Fi类型
    priority=1 #优先级
    }

    network={
    ssid=”WiFi-B”
    psk=”12345678”
    key_mgmt=WPA-PSK
    priority=2
    scan_ssid=1 #如果需要连接隐藏Wi-Fi
    }
