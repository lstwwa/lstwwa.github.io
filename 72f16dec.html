FPGA

1、什么叫“综合”，一般综合包含哪些过程？

综合的目的是将多个模块化设计文件合并为一个网表文件，并使层次设计平面化（即展平）

2、简述“自顶向下”的设计流程。

自顶向下的设计采用功能分割的方法，从顶层设计开始，逐层向下将设计内容进行分块和细化。

3、EDA工具大致可以分为哪几个模块？各个模块的主要功能是什么？

EDA工具大致可以分为设计输入编辑器，仿真器，HDL综合器，适配器（或布局线器）及下载器5个模块。

  • ①编辑器：图形输入和HDL文本输入
  • ②仿真器：行为模型表达，电子系统的建模，逻辑电路的验证及门级系统的测试
  • ③HDL综合器:将硬件描述语言转化为硬件电路的重要工具
  • ④适配器：完成在目标系统器件上的布局布线
  • ⑤下载器:把电路设计结果下载到实际器件中

4、VHDL特点：

  • ①VHDL具有强大的功能，覆盖面广，描述能力强。
  • ②VHDL有良好的可读性。
  • ③VHDL具有良好的可移植性。
  • ④使用VHDL可以延长设计的生命周期。
  • ⑤VHDL支持对大规模设计的分解和已有设计的再利用。
  • ⑥VHDL有利于保护知识产权。

5、简述EDA技术设计电路的设计流程。

设计准备、设计输入（原理图，硬件描述语言，波形图）、

设计处理（优化，综合，适配，分割，布局，布线）、

器件编程，以及相应的功能仿真、时序仿真和器件测试。

  • 1D、不正确的：D-不同器件
  • 2B、q[3:0]：B-‘b1011
  • 3A、A<<8‘b11100011结果：A-’b0011000
  • 4A、A=8‘11100011，B=8’0001000则A&&B：A-‘b1
  • 5B、不属于赋值语句：B-if else
  • 6C、eda不包括：C-可编程逻辑器件
  • 7D、非合法标识符：D-1_ne_a
  • 8D、缩减运算：D-y=&a
  • 9A、a=0，b=0，c=1:A-b‘00
  • 10C、C-case属于赋值语句

数字信号处理

选择

  • 1A、时序傅立叶变换：A-共轭对称函数
  • 2A、连续抽样信号在：A-单位圆
  • 3C、不是IRR：C-频率抽样型
  • 4A、关于FFT错误：A-新的变换
  • 5B、线性卷积6，圆周卷积5:B
  • 6B、N点DFT附属乘法次数：B-N2方
  • 7D、窗函数错误的：D-不能用于高通滤波器
  • 8A、FIR和IRR不正确的：A-FIR采用递归结构
  • 9A、yita（n）的z变换：A-1
  • 10A、抽样信号通过：A-理想低通滤波器
  • 11A、N需满足条件：A-N>=M
  • 12D、x(n)=sin(30π/120)则N=：D=8
  • 13A、收敛域包括：A-单位圆
  • 14C、FIR幅度响应H(w)特点：C-w=0、2π偶对称，w=π奇对称

填空

  • 1、x（n）长度64，h（n）长度128，y(n)为 64+128-1=191点，如果采用基2FFT，则FFT至少256点
  • 2、三种常用低通滤波器：巴特沃什滤波器、切比雪夫滤波器、椭圆滤波器
  • 3、关系为Ω=w/T，映射变换关系Ω=（2/T）tan（w/2）或者w=2arctan（ΩT/2）
  • DFT表达式为////，时域的长度为N，间隔是-(2π/M)

判断

  • 1：T-有限长的N点DFT。。。
  • 2：T-用双线变换法。。。
  • 3：F-DIF具有离散性和非周期性
  • 4：F-一个信号序列，。。。
  • 5：F-一个线性时不变的离散。。。
  • 6：T-序列的傅立叶变换。。。
  • 7：T-有限长序列的N点。。。
  • 8：F-模拟信号也可以。。。
  • 9：T-已知某离散系统。。。
  • 10：T-一个线性时不变系统。。。

module  fenpin(clk_1HZ,res,clk);
input  clk;
input  res;
reg[5:0]  cnt;
output  clk_1HZ;
parameter  Period=10;
assign  clk_1HZ=(cnt==Period)?1:0;
always  @(psoedgr  clk,posedge  res)
begin
if(res)begin  cnt<=0;end
else
begin
if(cnt>=Period)cnt<=0;
else  cnt<=cnt+1;
end
end
endmodule

end
