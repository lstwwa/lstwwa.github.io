  • 设计输入编辑器：将需要设计的系统的功能和结构以图形或文本的形式表达。

  • 仿真器：让计算器根据一定的算法和仿真库对EDA设计进行模拟，以验证设计，排除错误。

  • 适配器：完成目标系统器件上的布局线。

  • 下载器：把电路设计下载到实际器件中，实现硬件设计。

第一章

EDA技术就是依靠功能强大的电子计算机，在EDA软件工具平台上，对以硬件描述语言HDL（Hardware Description Language）为系统逻辑描述手段完成的设计文件，自动地完成逻辑编译、化简、分割、综合、优化、仿真，直至下载到可编程逻辑器件CPLD/FPGA或专用集成电路ASIC（Application Specific Integrated Circuit）芯片中，实现既定的电子电路设计功能。

EDA技术的发展分三个阶段

  1. CAD
  2. CAE
  3. EDA

EDA设计流程

设计输入形式：

  1. 图形
  2. 文本
  3. 波形

设计处理主要包括

  1. 设计编译和检查
  2. 设计优化和综合
  3. 适配和分割
  4. 布局和布线
  5. 生成编程数据文件等过程。

设计输入完成之后，立即进行编译。在编译首先进行语法检验，然后进行设计规则检验，检查总的设计有无超出器件资源或规定的限制并将编译报告列出，指明违反规则和潜在不可靠电路的情况以供设计者纠正 。

设计优化主要包括面积优化和速度优化。

在适配和分割过程，确定优化以后的逻辑能否与CPLD或FPGA中的宏单元和I/O单元适配，然后将设计分割为多个逻辑小块形式映射到器件相应的宏单元中。

布局和布线工作是在设计检验通过以后由软件自动完成的，它能以最优的方式对逻辑元件布局，并准确地实现元件间的布线互连。

设计处理的最后一步是产生可供器件编程使用的数据文件。FPGA是生成位流数据文件(Bit-stream Generation ，简称BG文件)。

设计校验过程包括功能仿真和时序仿真。

器件编程需要满足一定的条件，如编程电压、编程时序和编程算法等。在系统的可编程器件（ISP-PLD）则不需要专门的编程器，只要一根与计算机互连的下载编程电缆就可以了。

VHDL的特点

  1. VHDL具有强大的功能，覆盖面广，描述能力强。
  2. VHDL有良好的可读性。它可以被计算机接受，也容易被读者理解。
  3. VHDL具有良好的可移植性。
  4. 使用VHDL可以延长设计的生命周期。
  5. VHDL支持对大规模设计的分解和已有设计的再利用。
  6. VHDL有利于保护知识产权。

Verilog HDL可以用来进行各种层次的逻辑设计，也可以进行数字系统的逻辑综合、仿真验证和时序分析。

Verilog HDL适合算法级、寄存器传输级、逻辑级、门级和版图级等各个层次的电路设计和描述。

Verilog HDL最大优点是设计与工艺无关性，只需要根据系统设计的要求施加不同的约束条件，即可设计出实际电路。

AHDL是一种模块化的硬件描述语言，它完全集成于Altera公司的MAX+plusII的软件开发系统中。

可编程逻辑器件（PLD）是一种半定制集成电路，在其内部集成了大量的门和触发器等基本逻辑单元电路，用户通过编程来改变PLD内部电路的逻辑关系或连线，就可以得到所需要的设计电路。

采用PLD进行的数字系统设计，是基于芯片的设计或称之为“自底向上”的设计，硬件描述语言产生了目前最常用且称为“自顶向下”的设计法。

EDA工具大致可以分为设计5个模块等

  1. 输入编辑器
  2. 仿真器
  3. HDL综合器
  4. 适配器（或布局布线器）
  5. 下载器

Verilog HDL的语句包括以下等类型

  1. 赋值语句
  2. 条件语句
  3. 循环语句
  4. 结构说明语句
  5. 编译预处理语句

每一类语句又包括几种不同的语句。在这些语句中，有些语句属于顺序执行语句，有些语句属于并行执行语句。

用assign语句建模
assign语句一般适合对组合逻辑进行赋值，称为连续赋值方式。

1位全加器

module adder1(sum,cout,a,b,cin);
input         a,b,cin;
output     sum,cout;
assign     {cout,sum} = a+b+cin;
endmodule

用元件例化（instantiate）方式建模

3输入端与门 

and    myand3(y,a,b,c);
  • 用always块语句建模

always块语句可以产生各种逻辑，常用于时序逻辑的功能描述。一个程序设计模块中，可以包含一个或多个always语句。程序运行中，在某种条件满足时，就重复执行一遍always结构中的语句。

  • 用initial块语句建模
  • initial块语句预always语句类似，不过在程序中它只执行1次就结束了。

Verilog HDL程序设计模块的基本结构小结：

  1. Verilog HDL程序是由模块构成的。每个模块的内容都是嵌在module和endmodule两语句之间，每个模块实现特定的功能，模块是可以进行层次嵌套的。
  2. 每个模块首先要进行端口定义，并说明输入（input）、输出（output）或双向（inout），然后对模块的功能进行逻辑描述。
  3. Verilog HDL程序的书写格式自由，一行可以一条或多条语句，一条语句也可以分为多行写。
  4. 除了end或含end（如endmodule）语句外，每条语句后必须要有分号“；”。
  5. 可以用/……/或//……对Verilog HDL程序的任何部分作注释。一个完整的源程序都应当加上需要的注释，以加强程序的可读性。

Verilog HDL的空白符包括空、tab符号、换行和换页。空白符如果不是出现在字符串中，编译源程序时将被忽略。

位宽表示数字对应的二进制数的位数宽度；
进制符号包括

  1. b或B（表示二进制数）
  2. d或D（表示十进制数）
  3. h或H（表示十六进制数）
  4. o或O（表示八进制数）

例如

  1. 8’b10110001 //表示位宽为8位的二进制数
  2. 8’hf5 //表示位宽为8位的十六进制数

关键字是Verilog HDL预先定义的单词，它们在程序中有不同的使用目的。例如，module和endmodule来指出源程序模块的开始和结束；用assign来描述一个逻辑表达式等。

定义标识符时应遵循如下规则

  1. 首字符不能是数字。
  2. 字符数不能多于1024个。
  3. 大小写字母是不同的。
  4. 不要与关键字同名。

变量分为

  1. 网络型（nets type）
  2. 寄存器型（register type）

网络型变量是输出值始终根据输入变化而更新的变量，它一般用来定义硬件电路中的各种物理连线。

寄存器型变量是一种数值容器，不仅可以容纳当前值，也可以保持历史值，这一属性与触发器或寄存器的记忆功能有很好的对应关系。

reg型变量定义的关键字是reg，定义格式如下

reg [位宽]     变量1,变量2,…,变量n;

门级描述
用于门级描述关键字包括：

  1. not（非门）
  2. and（与门）
  3. nand（与非门）: nand nand2(y,a,b); //2输入端与非门
  4. or（或门）
  5. nor（或非门）
  6. xor（异或门）: xor myxor(y,a,b); //异或门
  7. xnor（异或非门）
  8. buf（缓冲器）: bufif0 mybuf(y,a,en); //低电平使能的三态缓冲器
  9. bufif1、bufif0、notif1、notif0等各种三态门。

Verilog HDL的行为级描述是最能体现EDA风格的硬件描述方式，它既可以描述简单的逻辑门，也可以描述复杂的数字系统乃至微处理器；既可以描述组合逻辑电路，也可以描述时序逻辑电路。

module rlshift8(q,d,lod,clk,clr,s,dir,dil);
  input [7:0]     d;
  input         lod,clk,clr,s,dir,dil;
  output [7:0]     q;
  reg  [7:0]     q;
always @(posedge clk)
    begin
    if (~clr)    q = 'b00000000;
      else if (lod)    q = d;
         else if (s) begin
                q = q >> 1;        --实现右移操作
            q[7] = dir; end
