EDA（FPGA）复习资料

  • FPGA技术及应用考试时间：2021年5月29日（周六下午）15：00–17：00

1.一般把EDA技术的发展分为三个阶段：

  1. CAD(早期)
  2. CAE(中期)
  3. EDA(现代)

2.EDA设计流程包括4个步骤：

  1. 设计准备
  2. 设计输入
  3. 设计处理
  4. 器件编程

3.设计输入方式分为三种方式：

  1. 图形输入
  2. 文本输入
  3. 波形输入

4.Verilog HDL与VHDL都是用于电路设计的硬件描述语言，不同在于：

Verilog HDL 推出时间早、应用历史长、设计群体广泛、设计资源丰富，并且其是由C语言演化而来。

5.EDA工具大致可分为5个：

  1. 设计输入编辑器
  2. 仿真器
  3. HDL综合器
  4. 适配器
  5. 下载器

6.Verilog HDL 有哪些主要特点？

  1. 可以用来进行各种层次的逻辑设计，也可以进行数字系统的逻辑综合、仿真验证和时序分析。
  2. 适合各个层次的电路设计和描述。
  3. 设计与工艺无关。
  4. 设计资源丰富。

7.简述在PLD没有出现以前，传统的数字系统的“积木式”过程？

  1. 先由器件搭成电路板，再由电路板搭成系统。

8.EDA工具大致分为那几个模块?各模块的主要功能？

  • EDA工具大致可分为5个：
  1. 设计输入编辑器
  2. 仿真器
  3. HDL综合器
  4. 适配器
  5. 下载器

  • 设计输入编辑器：将需要设计的系统的功能和结构以图形或文本的形式表达。

  • 仿真器：让计算器根据一定的算法和仿真库对EDA设计进行模拟，以验证设计，排除错误。

  • HDL综合器：将硬件描述语言转化为硬件电路的作用。

  • 适配器：完成目标系统器件上的布局线。

  • 下载器：把电路设计下载到实际器件中，实现硬件设计。
