卸载与安装系统应用

以下有两种常用的方式来进行对系统软件的卸载和停用

  • 停用软件

      adb shell pm disable-user +APPpackagename

  • 启用

      adb shell enable +APPpackagename

  • 卸载软件

      adb shell pm uninstall --user 0 +APPpackagename

  • 安装

      adb shell pm install-existing --user 0 +APPpackagename

他们有以下区别

  • 禁用的话接口会依然存在，但如果卸载的话就不会有接口

    例如华为的下拉搜索，使用禁用的话依然保留下拉动画

  • 某些情况下卸载反而会比停用更加安全

    例如小米禁用或者卸载应用商店后会卡米，但是禁用状态下无法进入ADB调试而卸载则可以进入ADB

您的AOSP基准版本号是
adb shell getprop ro.build.version.release

状态栏图标精简
adb shell am start -n com.android.systemui/com.android.systemui.DemoMode

修改熄屏时间（帮助您修改省电模式下的熄屏时间）
adb shell settings put system screen_off_timeout “%time%”

弹窗出现的过渡动画

adb shell settings put global window_animation_scale 1

是页面切换的过渡动画

adb shell settings put global transition_animation_scale 1

是软件打开的过渡动画

adb shell settings put global animator_duration_scale 1
